## New topics

- The linalg.norm complex issue is still being debugged.

- Array API ruff migration rule to auto-translate some things (e.g. dot to tensordot?).  Just for awareness: github.com/astral-sh/ruff/pull/24005

- Discussion about sorting API
  - Some tendency that maybe just shifting headers/making those nice in quaddtype might be a path.
 
- Issue with `arr.resize(..., refcheck=True)` being now _very_ strict.
  - Keeping the more strict version with better docs/error seems like the right choice.
